# Team Task Hub – Skilaverkefni 2

## How to run the project

### Install dependencies:

```bash
npm install
```

### Run the app

```bash
npm run dev
```

## Run Tests

### Vitest

```bash
npm run test:run
```

### Storybook

```bash
npm run storybook
```

### Cypress

```bash
npm run e2e
```

## Bug fixes

Two bugs were found and fixed during the testing work:

- Editing a task opened the form in add mode instead of edit mode.
- Deleting a project did not remove its related tasks from state/localStorage.
